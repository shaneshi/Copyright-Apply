# {{software_name}} 安装手册

## 1. 环境准备
- **操作系统**：Linux Server
- **JDK版本**：JDK 1.8+
- **Web服务器**：Nginx 1.20+
- **工具**：VSCode (用于代码调试)

## 2. 后端安装说明（通用 Java 流程）
1. 将部署包上传至 Linux 服务器目录。
2. 配置 `JAVA_HOME` 环境变量。
3. 执行 `java -jar server.jar` 启动后端服务逻辑。

## 3. 前端部署流程
1. 将 `process/` 目录下的 HTML 静态资源拷贝至 Nginx 的运行目录。
2. 修改 Nginx 配置文件，指向 `index.html`。
3. 重载 Nginx：`sudo nginx -s reload`。

## 4. 访问测试
在 Linux 浏览器中输入 `http://localhost:8080` 即可查看页面。
